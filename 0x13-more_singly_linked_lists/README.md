# Project: 0x13. C - More singly linked lists

> More exercises about when and why using linked lists vs arrays and how to build and use linked lists.

Files in this folder:

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 0-print_listint.c

Function that prints all the elements of a **listint_t** list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 1-listint_len.c

Function that returns the number of elements in a linked **listint_t** list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 2-add_nodeint.c

Function that adds a new node at the beginning of a **listint_t** list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 3-add_nodeint_end.c

Function that adds a new node at the end of a **listint_t** list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 4-free_listint.c

Function that frees a **listint_t** list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 5-free_listint2.c

Function that frees a **listint_t** list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 6-pop_listint.c

Function that deletes the head node of a **listint_t** linked list, and returns the head node&rsquo;s data (n).

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 7-get_nodeint.c

Function that returns the nth node of a **listint_t** linked list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 8-sum_listint.c

Function that returns the sum of all the data (n) of a **listint_t** linked list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 9-insert_nodeint.c

Function that inserts a new node at a given position.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 10-delete_nodeint.c

Function that deletes the node at index **index** of a **listint_t** linked list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 100-reverse_listint.c

Function that reverses a **listint_t** linked list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 101-print_listint_safe.c

Function that prints a **listint_t** linked list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 102-free_listint_safe.c

Function that frees a **listint_t** list.

#### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 103-find_loop.c
