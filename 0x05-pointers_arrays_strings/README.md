# Project: 0x05. C - Pointers, arrays and strings

```Firs steps with pionters```

Files in this project:

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 0-reset_to_98.c

Function that takes a pointer to an **int** as parameter and updates the value it points to to **98**.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 1-swap.c

Function that swaps the values of two integers.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 2-strlen.c

Function that returns the length of a string. FYI: The standard library provides a similar function: **strlen**. Run **man strlen** to learn more.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 3-puts.c

Function that prints a string, followed by a new line, to **stdout**. FYI: The standard library provides a similar function: **puts**. Run **man puts** to learn more.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 4-print_rev.c

Function that prints a string, in reverse, followed by a new line.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 5-rev_string.c

Function that reverses a string.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 6-puts2.c

Function that prints every other character of a string, starting with the first character, followed by a new line.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 7-puts_half.c

Function that prints half of a string, followed by a new line.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30">  8-print_array.c

Function that prints **n** elements of an array of integers, followed by a new line.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 9-strcpy.c

Function that copies the string pointed to by **src**, including the terminating null byte (**\0**), to the buffer pointed to by **dest**.FYI: The standard library provides a similar function: **strcpy**. Run **man strcpy** to learn more.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 100-atoi.c

Function that convert a string to an integer.FYI: The standard library provides a similar function: **atoi**. Run **man atoi** to learn more.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> 101-keygen.c

Program that generates random valid passwords for the program **101-crackme**.

### <img src="https://i.imgur.com/UCWcsWc.png" alt="Logo C" height="30"> holberton.h

Library of functions in this folder.

### <img src="https://i.imgur.com/s1rXGpW.png" alt="Logo C" height="30"> _putchar.c

Function that prints a char in stdio, without use stdio library.
